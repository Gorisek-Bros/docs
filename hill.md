# Hill

::: tip
The root element of your XML hill.
:::

| Name    | Type   | Allowed values         | Description                                             |
| ------- | ------ | ---------------------- | ------------------------------------------------------- |
| version | String | DSJ4-1.5.0, DSJ4-1.7.0 | Minimum DSJ4 version where the hill is supposed to work |
| author  | String |                        | Hill author(s)                                          |
